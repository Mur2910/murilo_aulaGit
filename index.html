O que é o Git

Git é uma ferramenta que ajuda você a controlar as mudanças nos seus arquivos. É ótimo para trabalhar em equipe, permitindo que várias pessoas façam alterações sem bagunçar tudo.

O que é o GitHub

GitHub é como uma rede social para programadores. Ele permite que você armazene seus projetos online e colabore com outras pessoas. É onde você pode compartilhar seu código e ver o que os outros estão fazendo.

Diferenças entre Git e GitHub

Git é a ferramenta que você usa no seu computador para gerenciar versões dos seus arquivos. Já o GitHub é um site onde você pode armazenar esses arquivos e colaborar com outras pessoas.

Comandos Básicos do Linux

Aqui estão alguns comandos básicos do Linux que você vai achar úteis:

  • cd nome_da_pasta - Entrar em uma pasta.
  • ls nome_da_pasta - Listar o conteúdo de uma pasta.
  • mkdir nome_da_pasta - Criar uma nova pasta.
  • rm -r nome_da_pasta - Excluir uma pasta.
  • mv nome_atual_do_arquivo nome_novo_do_arquivo - Renomear um arquivo.
  • mv nome_do_arquivo caminho_da_pasta/ - Mover um arquivo para outra pasta.
  • cp nome_do_arquivo caminho_da_pasta_de_destino/ - Copiar um arquivo para outra pasta.

Como criar um repositório no GitHub

Para criar um repositório, entre na sua conta do GitHub, clique em "New" e siga as instruções. É rápido e fácil!

Como ligar a um projeto existente na máquina

Para começar a usar o Git em um projeto que já existe, abra o terminal na pasta do projeto e digite git init para inicializar o repositório.

Como gerar uma versão com Git

Para marcar uma nova versão do seu projeto, use git tag -a v1.0 -m "Versão 1.0". Assim, você pode voltar a essa versão quando precisar.

Como enviar da máquina para o GitHub

Depois de fazer suas alterações e salvar com um commit, use git push origin main para enviar tudo para o GitHub.
